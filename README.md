# IT1214-Student-Grade-System

## Project Purpose
This Java console application allows teachers to manage student records and marks.

## Features
- Add student
- Display all students
- Search student by ID
- Calculate average marks

## How to Run

Compile:
  javac Main.java
Run:
  java Main
